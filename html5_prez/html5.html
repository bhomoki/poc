Goals

  • Semantics
  • Separation of concerns
  • UX
  • No strict rules
  • XHTML too document centric

New elements

  • Tags (address, article, aside, div, footer, header, hgroup, layer, main, nav, section)
  • Input types (number, color, range, date, month, week, time, datetime, datetime-local, email, search, tel, url)
  • Input attributes (max, min, pattern, required, step)
  • data-* (custom data attributes)
  • Case-insensitive
